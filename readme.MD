# Definición
El trabajo práctico final de la materia Computación, algoritmos y estructuras de datos de la Maestría en Inteligencia artificial (MIA) de la Universidad de Buenos Aires (UBA) consiste en la resolución de una serie de ejercicios relacionados con la temática vista.

# Ejercicios

## Ejercicio 1
Implementar una máquina de Turing que puede sumar dos números binarios. Se desarrolló una máquina con 3 cintas. A su vez, se investigó y encontró información para realizar una versión con una única cinta, por lo que se implementó y comparó con la desarrollada.
